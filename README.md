# BharatMind AI — Enterprise

**India's AI Business Operating System**

BharatMind is a unified AI copilot that helps Indian SMBs run their entire business — finance, GST compliance, inventory, and customer communication — through one intelligent platform, in their own language.

## What it does

- **AI Copilot** — Bilingual business assistant trained on localized commerce guidelines; reads PDFs, contracts, and receipts instantly
- **Finance Suite** — Autonomous ledger audits, bank reconciliation, and mismatch detection with 100% security compliance
- **GST Intelligence** — Real-time compliance checks, cross-verifies GSTR-1 & 3B, predicts audit mismatch risk
- **Voice AI** — Natural-cadence calling agents for payment reminders and order follow-ups in 4 regional languages
- **Deep Integrations** — Native sync with Tally Prime, GST Portal, Shopify, WhatsApp Business, Zoho Books, Razorpay, MS Excel, Power BI, Google Analytics, and Microsoft 365 — no migration needed

## Why BharatMind

Most Indian SMBs run on a patchwork of Tally, Excel, and WhatsApp with no unified intelligence layer. BharatMind connects to tools they already trust and adds an AI reconciliation and automation layer on top — turning manual entry and compliance chasing into a single conversational interface.

## Tech Stack

- Frontend: TypeScript, Vite, React
- AI: Multi-LLM pipeline (Claude, Gemini, Sarvam APIs)
- Infra: Azure + Firebase backend

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Set `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app: `npm run dev`
